## 🧠 What this tool does

This application acts as a bridge between your needs and the Codex engine. Many users find it difficult to write prompts that lead to exact outputs. You might try to explain a goal, but the system produces random code. This builder solves that problem. It organizes your input into fields that the AI expects. You do not need to write code to use this tool. You answer questions and pick options from menus. The builder turns these inputs into a final, clean prompt.

### Features
- Preset templates for standard coding tasks.
- Validation checks to ensure prompt quality.
- Audit history of every prompt you build.
- Export functions for your preferred text editor.
- Searchable database of past project goals.

## 🚀 Running your first session

1. Open the Goal Prompt Builder from your desktop icon.
2. Click "New Project" on the main screen.
3. Select a category from the provided list. Choose the category that fits your specific goal.
4. Enter your requirements into the text boxes. Use plain words to describe what you want the code to do.
5. Review the preview pane on the right side of the screen. This shows the final instruction text.
6. Click "Copy to Clipboard" to move the text to your editor.
7. Paste your new prompt directly into your AI interface.

## 🔍 Understanding the interface

The dashboard features three primary areas:

1. The Workspace: This is where you enter details about your task. 
2. The Editor: This area allows you to fine-tune the phrasing of your goals.
3. The Audit Log: This keeps a record of all prompts you created. Use this if you need to revisit an old task for a review or to check for compliance.

## 🛡️ Audit-friendly prompts

You might work in a field where you must document why you chose certain code styles. The Goal Prompt Builder simplifies this. Every prompt you save includes a timestamp and the parameters you selected. You can attach these files to your reports. Auditors can see exactly how you guided the system to reach the final result. This visibility reduces questions about your process.

## 📝 Troubleshooting common issues

If the application fails to open:
- Restart your computer. 
- Ensure you installed the latest .NET Runtime version from the Microsoft website.
- Check your antivirus software to make sure it does not block the application. 
- Check that your user account has permission to run third-party software.

If the prompt does not return the expected result:
- Review your input fields for missing details.
- Add more context to your task description.
- Try selecting a different template in the application.

## 📈 Frequently asked questions

Does this tool connect to the internet?
The tool uses a local interface on your computer. It only sends data to the AI engine when you request a prompt execution.

Is my data private?
Yes. All your project files stay on your hard drive until you choose to copy the output.

Can I share my prompts with team members?
You can save your workspace files and share them with colleagues. They can open the files in their own installation of the builder.

Does this work on Mac or Linux?
The current version only supports Windows. 

Will there be updates?
New templates and features arrive through the regular release cycle. Check the repository for the latest version.
